# jamesskowyra.com

Skowyra development website, built on [Drupal 11](https://www.drupal.org/drupal-11).

## Requirements

- [Composer](https://getcomposer.org/) 2.x
- [DDEV](https://ddev.readthedocs.io/) (recommended for local development)
- [OrbStack](https://orbstack.dev/) (Docker provider)
- PHP 8.3+

## Local Development Setup

### Using DDEV with OrbStack (recommended)

1. **Install OrbStack**: Download and install [OrbStack](https://orbstack.dev/). It serves as the Docker provider and offers native Linux filesystem performance on macOS — no Mutagen sync needed.

2. **Install DDEV**: Follow the [DDEV installation guide](https://ddev.readthedocs.io/en/stable/users/install/ddev-installation/).

3. **Clone the repository** and install dependencies:
   ```bash
   git clone https://github.com/skowyra/jamesskowyra.com.git
   cd jamesskowyra.com
   ddev start
   ddev composer install
   ```

4. **Install Drupal**:
   ```bash
   ddev drush site:install --account-name=admin --account-pass=admin -y
   ```

5. **Open the site**:
   ```bash
   ddev launch
   ```

### Without DDEV

1. Install dependencies:
   ```bash
   composer install
   ```

2. Configure your web server to point to the `web/` directory.

3. Copy `web/sites/default/default.settings.php` to `web/sites/default/settings.php` and configure your database connection.

4. Install Drupal via the browser at `/install.php` or using Drush:
   ```bash
   vendor/bin/drush site:install -y
   ```

## Directory Structure

```
├── composer.json      # Project dependencies
├── composer.lock      # Locked dependency versions
├── .ddev/             # DDEV local development configuration
├── web/               # Drupal webroot (generated by composer install)
│   ├── core/          # Drupal core (managed by Composer)
│   ├── modules/
│   │   ├── contrib/   # Contributed modules (managed by Composer)
│   │   └── custom/    # Custom modules
│   ├── themes/
│   │   ├── contrib/   # Contributed themes (managed by Composer)
│   │   └── custom/    # Custom themes
│   └── sites/
│       └── default/
└── vendor/            # PHP dependencies (managed by Composer)
```

## Contributing

- Custom modules go in `web/modules/custom/`
- Custom themes go in `web/themes/custom/`
- Add contributed modules/themes via Composer: `composer require drupal/MODULE_NAME`
